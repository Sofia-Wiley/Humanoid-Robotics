# Humanoid Robotics Simulations (pyCub)

Solutions for the practical exercises in CTU's Humanoid Robotics course
(Spring 2026), implemented in Python

## Exercises

### 1. Vision-Based Grasping
File: `grasping.py`

Detects a target object using the simulator's onboard cameras, plans an
approach trajectory in Cartesian space, closes the hand around the object,
and lifts it off the table. Switches what hand to use based on ball location.

### 2. Gaze Tracking
File: `gaze_tracking.py`

Commands the iCub's head and eye joints to follow a moving target in real
time, keeping the target centered in the camera frame using visual feedback.
